Google summer of Code Student 2019 with JdeRobot

    Topic: Migration of JdeRobot tools to ROS2

    ROS has changed the world of Robotics, now we need not write every single library for a new project which has already been implemented before. Researchers and developers can work on top of developed tools and libraries. ROS2 has brought good changes with it, including all features of ROS along with new ones to address shortcomings in earlier version. To make use of those characteristics it is important to migrate our tools and drivers. This project aims to adapt drivers and some crucial JDERobot tools to make them ROS2 compatible.

    Link to Blog: https://theroboticsclub.github.io/colab-gsoc2019-Pankhuri_Vanjani/2019-08-23-final-report/

Summer Research Internship at Indraprastha Institute of Information Technology Delhi

    Topic : Multi-UAV-collision-avoidance

    UAV’S have paved their way into different fields of applications like surveillance, communication. Multi-uav’s have great importance in such areas. Coordination of vehicles is one of the critical tasks under which path planning is looked upto. For this foremost condition is collision avoidance maneuver. Avoiding collision with static as well as dynamic objects. In this project we have worked on collision avoidance with different drone(dynamic obstacle).

    Link of presentation: https://docs.google.com/presentation/d/1pUHR8dj-EFrFmUQMYCmkI7UOELV1b3POl7ajcajcSE8/edit#slide=id.p3

    Link to github repository: https://github.com/pankhurivanjani/Multi-UAV-collision-avoidance
